# Tailwind UI Component – Footer

A reusable and responsive UI component built with **Tailwind CSS**.  
This component is lightweight, customizable, and easy to integrate into any Tailwind project.

## Setup

1. Run this command to install package.json, package-lock.json and node_modules in 'Footer' folder:
   ```bash
   cd UI-Components---Tailwind-CSS/Footer/
   npm init -y
   npm install node
   ```
    
2. Run this command in your project directory [cd Project-Directory] to start making changes in UI Component:
   ```bash
   npx tailwindcss -i ./src/input.css -o ./src/output.css --watch  
   ```
   After running this command it will occupy your terminal as it has to monitor changes in 'index.html' as css is being
   created in background in 'output.css' as we utilize classes in 'index.html'.
